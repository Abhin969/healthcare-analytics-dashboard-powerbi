# Healthcare Analytics Dashboard
📌 Overview

This project presents an interactive Power BI dashboard designed to analyze healthcare data including patient trends, hospital performance, and billing insights.

🎯 Key Features

Patient demographics analysis
Hospital performance tracking
Doctor performance insights
Insurance provider distribution
Billing analysis
Interactive slicers and filters

📂 Dataset Includes

Patient Name, Age, Gender
Medical Condition
Hospital & Doctor
Admission & Discharge Dates
Billing Amount
Insurance Provider

🧮 Tools & Technologies

Excel
Power BI
DAX
Power Query

🚀 Key Insights

Balanced gender distribution
High emergency admissions
Top hospitals dominate patient intake
Chronic diseases are common
Billing trends align with patient volume
